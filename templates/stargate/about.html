About Project

The Stargate SG-1 Fanpage website was created as a competition project for ninth-grade students of elementary schools who want to apply to study at the Industrial High School of Electrical Engineering SPŠE Hálova, Bratislava, Slovakia.

This project belongs to the category of programming. I decided to create it using the Python and JavaScript programming languages ​​as well as HTML and CSS. I built this website from scratch from design to implementation of the entire project using my current skills and, of course, googling here and there.

I chose the theme of the Stargate SG-1 TV series to create this website because I am a big fan of the series. It seems that the series will probably not continue in the future, which is a pity, so I decided to add some quotes here to recall some moments.
